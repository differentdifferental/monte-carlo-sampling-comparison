## 📖 项目概述

本项目系统比较了四种经典蒙特卡洛抽样方法在谐振子系统信息熵计算中的性能表现：

- **Box-Muller变换法**：精确数学变换方法
- **接受-拒绝法**：通用性强的包络抽样方法  
- **Metropolis算法**：马尔可夫链蒙特卡洛方法
- **CLT近似法**：基于中心极限定理的快速近似方法

通过构建多维度量化评估体系（精确性、效率、稳定性、收敛性、通用性），为蒙特卡洛方法在实际科学计算问题中的选择提供数据驱动的决策依据。

## 📊 核心发现（基于实验数据）

| 方法 | 平均相对误差 | 平均执行时间 | 稳定性（标准差） | 推荐场景 |
|------|-------------|-------------|------------------|----------|
| Box-Muller变换 | 0.109% | 1.30 ms | 0.003655 | 精确高斯分布抽样 |
| 接受-拒绝法 | 0.100% | 7.86 ms | 0.002971 | 高精度通用分布抽样 |
| Metropolis算法 | 0.383% | 4.21 ms | 0.007594 | 复杂/高维分布抽样 |
| CLT近似法 | 0.205% | 3.41 ms | 0.003033 | 快速工程近似 |

> **注**：执行时间受运行环境（CPU性能、系统负载等）影响，表中为典型参考值。所有统计结果（误差、稳定性等）基于固定随机种子（seed=42）确保可重现性。

## 🚀 快速开始

### 环境要求
- MATLAB R2016b 或更高版本
- 建议内存：4GB 以上

### 运行步骤
1. 克隆或下载本仓库
2. 在MATLAB中打开项目文件夹
3. 运行主程序：
   ```matlab
   main_program
   ```
4. 查看 `output/` 文件夹中的结果

### ⚠️ 重要说明
1. **结果可重现性**：程序使用固定随机种子（seed=42），**统计结果（误差、标准差等）完全可重现**
2. **执行时间**：程序执行时间受运行环境（CPU频率、系统负载、MATLAB版本）影响，不同机器上的测量值可能略有差异
3. **输出目录**：首次运行时自动创建 `output/` 文件夹，包含所有图表和实验报告

## 📊 实验参数

| 参数 | 值 | 说明 |
|------|----|------|
| 样本数 | 50,000 | Box-Muller、接受-拒绝、CLT方法的样本数 |
| MCMC总样本数/燃烧期 | 60,000/10,000 | Metropolis算法参数 |
| 温度范围 | [0.5, 1.0, 1.5, 2.0, 2.5, 3.0] | 对应方差σ²=T的正态分布 |
| 重复次数 | 10 | 稳定性评估重复实验次数 |
| 随机种子 | 42 | 确保统计结果可重现 |

## 📈 性能评估维度

1. **精确性**：与理论熵值的相对误差
2. **效率**：单次执行时间（毫秒级）
3. **稳定性**：重复实验的标准差
4. **收敛性**：误差随样本量的变化规律
5. **通用性**：方法对复杂分布的适应性

## 🔬 方法特点总结

| 方法 | 优点 | 缺点 | 最佳适用场景 |
|------|------|------|-------------|
| **Box-Muller变换** | 精度高、效率最佳、样本独立 | 仅适用于正态分布 | 精确高斯分布抽样 |
| **接受-拒绝法** | 精度最高、稳定性好、通用性强 | 效率较低、接受率影响性能 | 高精度通用分布抽样 |
| **Metropolis算法** | 通用性最强、处理复杂分布 | 样本相关、收敛慢、精度较低 | 复杂/高维分布抽样 |
| **CLT近似法** | 实现简单、效率高 | 精度有限、仅近似正态分布 | 快速工程近似 |
